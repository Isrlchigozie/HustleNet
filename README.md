
# HustleNet (v1 - Firebase Edition) 

> A campus freelance platform for students to showcase their skills, get booked, and earn — right from school.



## 🌐 Live Demo

> Hosted on Netlify  
> [https://absu-hustlenet.netlify.app](https://absu-hustlenet.netlify.app)


## About This Version

**HustleNet v1** is the **first Minimum Viable Product (MVP)** of a student-to-student freelance platform built for **Abia State University (ABSU)**.  
The goal: help students **monetize their skills** and connect with others who need help — whether it's design, writing, tutoring, or any hustle.

This version was built entirely using **Firebase + JavaScript** for rapid prototyping and proof of concept.


##  Core Features

- ✅ Students can register as either a **buyer** or a **seller**
- ✅ Sellers can list their skills and services
- ✅ Buyers can browse talents and book services
- ✅ Firebase Authentication + Firestore for data handling
- ✅ Clean, mobile-friendly UI
- ✅ Netlify hosting for speed and ease



##  Tech Stack

| Frontend         | Backend / DB      | Hosting         |

##  Privacy

All data is securely stored using **Firebase Firestore**.  
Authentication is powered by Firebase Auth.  
No personal data is shared outside the app.
